Flagship Project

Physics-informed 3D reconstruction from low-dose TEM

A research and software project on recovering 3D graphene structure from single low-dose TEM frames using preprocessing, statistical initialization, forward simulation, simulated annealing, and MD-based regularization.

Overview

A reconstruction problem where imaging physics matters

The core problem is to recover 3D atomic coordinates from a noisy 2D low-dose TEM image. This is a severely ill-posed inverse problem: the projection compresses depth information, the signal-to-noise ratio is low, and many candidate structures can look similar in image space.

My work approaches that problem as a staged reconstruction system rather than a single model. The implementation combines image conditioning, atom initialization, projected-charge-density z estimation, physics-based forward simulation, and structure refinement constrained by molecular geometry.

Positioning: this is not a generic ML project. It is computational imaging and scientific computing work grounded in experimental data, forward models, optimization, and reproducible software.

Pipeline

Four technical stages

01

Preprocessing

Flat-field correction, dead-pixel interpolation, BM3D denoising, and temporal averaging prepare low-dose TEM frames for reconstruction.

  • Compared BM3D, K-SVD, and U-Net during denoising studies
  • Selected BM3D for the public pipeline because it avoids training-data dependence
02

Initialization

Detect bright hole centers, infer atom x-y positions from hole-triplet geometry, add selective MAP-based fallback in difficult regions, estimate z with projected charge density, then smooth with LOWESS.

  • Geometry-based initialization plus defect-region recovery
  • Structured NPZ/CSV/JSON handoff into refinement stages
03

Optimization

Stage 3 couples simulated annealing with forward simulation and structural regularization. The codebase includes abTEM-based image simulation and LAMMPS adapters for MD-informed refinement.

  • Single-atom proposal moves for local structure refinement
  • Paper-aligned architecture verification on simulated graphene
04

Analysis

Reconstructed structures support downstream analysis of ripple dynamics, geometry, and structure-property behavior, turning image reconstruction into a usable scientific workflow.

  • Visual QC artifacts and reports for each stage
  • Simulation and validation utilities for reproducible comparisons

Results

Repository status and technical takeaways

Public repository

The open-source codebase documents simulated-data generation, staged validation, and paper-aligned architecture verification. On the current simulated Stage 2 validation case, the repository reports 568 matched atoms out of 640 with 0 extra detections and 0.62 px x-y RMSE.

Research context

The associated paper reports stronger end-to-end reconstruction performance, while the public repository is explicit about what has already been reproduced, what has been architecturally verified, and which longer production runs remain future work.

Engineering value

The strongest portfolio signal here is not just the scientific topic, but the system design: modular stages, validation harnesses, simulation utilities, structured outputs, and clear separation between mature and in-progress components.

Analysis

From reconstruction to interpretable structure analysis

Surface interpretation

LOWESS smoothing is used as part of the initialization workflow to convert noisy z estimates into a more coherent structural surface before refinement.

Structure-property analysis

The broader research program connects reconstructed geometry to downstream analysis of ripple structure and electronic behavior, helping link imaging outputs to physically meaningful material descriptors.
