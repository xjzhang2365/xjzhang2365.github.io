About

Solving hard imaging problems with algorithms that respect the data and the physics

I am a computational scientist trained at the intersection of experimental imaging, inverse problems, and algorithm development. My work focuses on recovering structure from noisy measurements, especially in settings where standard image processing is not enough and the underlying physics matters.

Across research and software projects, I build complete technical workflows: preprocessing, initialization, reconstruction, simulation, validation, and analysis. That includes translating research methods into maintainable Python code and designing pipelines that are inspectable, reproducible, and useful beyond a single experiment.

Featured Project

Physics-informed 3D reconstruction from low-dose TEM

Flagship PhD project on recovering 3D graphene structure from single low-dose transmission electron microscopy frames.

Inverse Problems Computational Imaging Scientific Software

This project addresses a severely ill-posed reconstruction problem: inferring 3D atomic coordinates from a noisy 2D projection. The pipeline combines low-dose image preprocessing, atomic initialization, projected-charge-density z estimation, simulated annealing, and molecular-dynamics-based structural regularization.

Problem
3D atomic reconstruction from single-frame low-dose TEM data.
Pipeline
Preprocessing, initialization, refinement, simulation, and validation.
Methods
BM3D, MAP-based detection, LOWESS, SA, abTEM, LAMMPS.
Outcome
Public repo documents staged validation on simulated graphene and paper-aligned architecture verification.
